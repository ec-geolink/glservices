RESTful
A RESTful API using the OpenSearch patterns will be established and exposed.
Linked Data API
A vocabulary and processing model for a configurable API layer intended to support the creation of simple RESTful APIs over RDF triple stores. https://github.com/UKGovLD/linked-data-api/blob/wiki/API_Deployment_Example.md.
The Web is the API
Approaches for using web architecture patterns to expose the graph for consumption. Particular attentuon will be payed to seeing how this approach can address issues of machine data harvesting as opposed to more fine grained application access.
